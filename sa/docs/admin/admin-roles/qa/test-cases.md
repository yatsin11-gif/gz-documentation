### TC-002: Форма создания администратора не сохраняется без выбранного раздела

- Status: Draft
- Type: Functional
- Priority: Medium
- Severity: Major
- Behavior: Negative
- Layer: E2E
- Suite: Admin Roles
- Related criteria: Создание администратора (супер-админ)
- Related Jira: `frontend-admin-roles-create-and-navigation.md`, `backend-admin-roles-rbac.md`
- Qase case: Не назначено
- Tags: `admin`, `roles`, `validation`
- Automation status: automation candidate

#### Preconditions

- Супер-админ залогинен в `gozap-admin`.

#### Test Data

- Логин, пароль, имя нового администратора без отмеченных разделов.

#### Steps

1. Action: Заполнить форму создания администратора, не отметив ни одного раздела, и попытаться сохранить.
   Expected result: Форма не сохраняется, отображается ошибка валидации.
2. Action: Отправить аналогичный запрос напрямую в backend API без указанных разделов.
   Expected result: Backend отклоняет создание администратора без хотя бы одного раздела.

#### Notes

- Покрывает и FE, и BE валидацию обязательного поля.

### TC-004: Backend отклоняет обращение администратора к неназначенному разделу

- Status: Draft
- Type: Security
- Priority: High
- Severity: Critical
- Behavior: Negative
- Layer: API
- Suite: Admin Roles
- Related criteria: Авторизация по разделам
- Related Jira: `backend-admin-roles-rbac.md`
- Qase case: Не назначено
- Tags: `admin`, `roles`, `security`
- Automation status: automation candidate

#### Preconditions

- Создан администратор с набором разделов, не включающим раздел «Партнёры».

#### Test Data

- Учётные данные администратора с ограниченным набором разделов.
- Эндпоинт раздела «Партнёры», не входящего в назначенный набор.

#### Steps

1. Action: Под учёткой администратора с ограниченным набором разделов обратиться напрямую к API раздела «Партнёры» (минуя FE-навигацию).
   Expected result: Backend отклоняет запрос и не возвращает данные раздела.
2. Action: Выполнить тот же запрос под учёткой супер-админа.
   Expected result: Запрос выполняется успешно, данные возвращаются.

#### Notes

- Точный код ответа (403/редирект) не зафиксирован в критериях (см. open question в feature-criteria.md).
