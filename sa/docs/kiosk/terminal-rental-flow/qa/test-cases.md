### TC-001: Основной flow аренды: номер телефона, SMS-код, карта, выдача PowerBank

- Status: Draft
- Type: Functional
- Priority: High
- Severity: Critical
- Behavior: Positive
- Layer: E2E
- Suite: Terminal Rental Flow
- Related criteria: Ввод номера и подтверждение SMS; Привязка карты и оплата
- Related Jira: `backend-terminal-rental-phone-confirmation-account.md`, `backend-terminal-rental-payment-and-dispense.md`, `frontend-terminal-rental-main-flow-screens.md`
- Qase case: Не назначено
- Tags: `kiosk`, `rental`, `payments`
- Automation status: manual

#### Preconditions

- Станция находится в состоянии покоя.
- Тестовый номер телефона и тестовая карта доступны в sandbox.

#### Test Data

- Номер телефона.
- Тестовая карта (sandbox платёжного провайдера).

#### Steps

1. Action: На экране покоя ввести номер телефона.
   Expected result: Станция отправляет SMS-код и показывает экран ввода кода.
2. Action: Ввести полученный SMS-код.
   Expected result: Код подтверждён, создан аккаунт по номеру телефона, показан экран привязки карты с инструкцией.
3. Action: Приложить тестовую карту для оплаты.
   Expected result: Оплата проходит успешно, карта сохраняется в аккаунте.
4. Action: Дождаться завершения выдачи.
   Expected result: Станция выдаёт PowerBank, отображается экран успеха.

#### Notes

- Платёжный провайдер терминала не подтверждён (см. coverage gaps).
